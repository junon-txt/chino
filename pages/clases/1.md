# Clase 1

Se trabajará con el libro HSK 1.

## Conversación - Texto (1.1)

### 01-1


??? note "A: 你好! "
    Nǐ hǎo!
    Hola!

??? note "B: 你好!"
    Nǐ hǎo!
    Hola!

#### Palabras nuevas

- 你 nǐ pron. (singular): vos
- 好 hǎo adj. bueno, bien 

### 01-2

??? note "A: 您好!"
    Nín hǎo
    Hola! (formal)

??? note "B: 你们好!"
    Nǐmen hǎo
    Hola!

#### Palabras nuevas

- 您 nín pron. (singular, formal): usted
- 你们 nǐmen pron. (plural): ustedes 

### 01-3

??? note "A: 对不起。"
    Duìbùqǐ.
    Perdón.

??? note "B: 没关系。"
    Méiguānxì.
    No hay problema.

#### Palabras nuevas

- 对不起 duìbùqǐ (v.): perdón
- 没关系 méiguānxì: no hay problema
